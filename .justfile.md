# Justfile Quick Reference

This project uses [just](https://github.com/casey/just) as a command runner for local development automation. The application is TypeScript (Node.js, ESM); commands run via `npm`/`npx tsx`/`tsc`/`node --test`.

## Installation

```bash
# macOS
brew install just node

# Linux
cargo install just
# or download node from https://nodejs.org/
```

**Note**: `npm install` installs dependencies (including tsx, typescript, and runtime deps).

## Quick Start

```bash
just              # show all commands
just ci           # run all CI checks locally (emulates GitHub Actions)
just quick-check  # fast check (tsc only)
just full-check   # full CI
```

## Most Common Commands

### Quality Checks

```bash
just ci           # tsc --noEmit + node:test + markdownlint
just typecheck    # TypeScript typecheck (tsc --noEmit)
just test         # node:test suite (tsx loader)
just markdownlint # markdown linting
just lint         # typecheck + markdownlint
```

### Testing

```bash
just test               # run all tests
just test-file FILE     # run a specific test file
```

### Setup & Installation

```bash
just setup          # npm install + pre-commit hooks
just install-deps    # npm install
just install-hooks   # pre-commit install
```

### Running

```bash
just dashboard      # TUI dashboard (npx tsx dashboard.ts)
just status         # system status
just scan REPO      # per-repo loop
just validate-config [FILE]  # validate config
just send-approval  # approve a waiting pr-loop
```

### Cleanup

```bash
just clean   # remove caches / local DB / logs
```

## Workflows

### Before Pushing

```bash
just ci      # run full CI locally (tsc + tests + markdownlint)
git push
```

## CI Emulation

`just ci` emulates the GitHub Actions workflow:

1. TypeScript typecheck (`tsc --noEmit`)
2. Unit tests (`node --import tsx --test tests/*.test.ts`)
3. Markdown linting

## Tips

- Run `just` (no args) to list all recipes.
- Run sibling `.ts` scripts with `npx tsx <script.ts>` or `node --import tsx <script.ts>`.
- Typecheck only what you need: `just typecheck`.
